# Hyu Ramadhan Planner

Welcome to the official page for **Hyu Ramadhan Planner** - A Flutter mobile application designed to help users track daily ibadah during Ramadhan with gamification, spiritual features, and interactive UI.

## About the App

**Hyu Ramadhan Planner** helps Muslims track their spiritual journey during Ramadhan with features including:

- **Ibadah Tracking**: Daily completion tracking for prayers, charity, and more
- **Gamification**: XP/Level system with 15+ achievement badges
- **Spiritual Features**: Prayer times, Quran tracker with calendar-based progress
- **Interactive UI**: Haptic feedback, celebrations, and animated widgets
